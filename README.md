# Rails API BoilerPlate

This is Rails API Boiler plate with DRY API pattern.

* Ruby version: 2.6.6

* System dependencies
  - Postgresql
  - Ruby
  - Bundler

* Configuration
  - bundle install
  - cp `.env.example` to `.env`

* Database creation
  - rake db:create 

* Database initialization
  - rake db:migrate
  
* How to run the test suite
  - rspec
